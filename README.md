## Mars Real Estate 

A mock real estate app that lets you post, update, and delete listings selling plots of land on Mars. 

![Screenshot] (./images/screenshot1.png?rew=true)

![Screenshot] (./images/screenshot2.png?rew=true)

#### Technologies Used
- Mongoose 
- MongoDB 
- Express
- EJS
- CRUD
- Node.js
- Passport
- OAuth
- JavaScript
- CSS
- Bootstrap
